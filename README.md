# Drawing with canvas

Small application for drawing shapes with HTML canvas API. check it out [here](https://guerrero25.github.io/drawing-canvas/)

# Features

- [x] Drawing square
- [x] Drawing filled squared
- [x] Drawing lines
- [x] Free drawing
- [ ] Saving as an image
- [ ] Color picker
- [ ] Line weight picker

# Developed with

- Typescript
- Test Driven Development
- React.js
